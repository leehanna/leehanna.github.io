1. Experience Strategy

Original Branding and Website Structure

There are many different categories of redistricting data grouped by years or decades. The current website structure requires users to click through multiple links to get to the page with the specific redistricting data by county. Statewide Database does not have branding.

Understanding Redistricting and Competitive Analysis

I first took the time to gain a basic understanding of what redistricting data is. Then, I researched how other websites offering redistricting data organized their websites and found them to also have complicated categorizations of the data.

Information Architecture

I proposed to offer users two different categorizations of the data. The main header navigation bar would be categorized first by census type and then have all associated files of each year, and the footer will be categorized just by type of data. I asked experts in the organization to help me restructure the website.

2. Visual Design

Logo Design

For a simple logo, I chose a serif font similar to the one being used in the the original website and created my own version of the California state image that is on the navigation on the right.

Header Navigation

To support a multi-level categorization in the header for the new sitemap, I opted for a large header similar to Amazon’s.

Footer

In the footer, I organized by the type of file, and had the contact form accessible from each page.

Homepage UI Design

In the homepage, I placed additional ways to search the website first since the main purpose of the website is to allow users to find the redistricting data they need. The categorical search function that I implemented using PHP is also available on every page via the side navigation bar. I then placed changing information, namely updates to the database and redistricting news, so that returning users can see new content.

Reflections

Designing a super intuitive navigation for Statewide Database was challenging since I did not have years of law background, and users in the target group were difficult to come by in my daily life. Unfortunately, even though I prepared the website for migration to the main server and left instructions before I left, it has still not been updated.
